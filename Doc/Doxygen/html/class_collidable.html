Detailed Description

This Class holds data used to manage the collision of a gamecomponent.

This class holds an size and positon to calculate its collision to any other collidable. any gamecomponent holding one wil be checked on collision if the hasMoved is set to true. Cause any component that has not moved can't collide. (this does not mean it can't be collided to. its just never the cause of collision.)
