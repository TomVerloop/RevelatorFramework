Detailed Description

This Singleton Manages and plays all music and soundeffects.

An empty class for future use.

This Abstract Class holds data used to manage the drawing of a gamecomponent.

This Class is Singleton defined. It holds an reference to all soundeffects and music. It makes sure only one song is played at a time. sound effects however can be played multiple at a time.
