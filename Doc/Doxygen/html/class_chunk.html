Detailed Description

This Class represents a small portion of a screen.

This Class represents a small portion of a screen. it makes sure nothing more is drawn than necessary. It handles its own draw and that of the neigboring chunks. and it holds all components that are at its location. further more it handles the vision and collision of the components in it.

Author
Tom Verloop
Version
1.0
Date
2014-2015
Bug:

anything with a vision greater that one chunk next to it is useless.

any component given to the next chunk will be updated twice.
